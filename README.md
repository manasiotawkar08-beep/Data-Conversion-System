 🔄 Data Converter System

This is a simple web application developed as an internship project. It converts data from one format to another, validates user input, and stores conversion records in a MySQL database.

## ✨ Features
- User Login
- Data Conversion
- Conversion History
- Delete Records
- Clean & Responsive UI


## 🛠️ Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

This project helped me improve my skills in web development, database management, and data transformation using PHP and MYSQL
